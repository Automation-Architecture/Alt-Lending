# Alt Lending - Signal-Based Lead Generation System

This repository contains the project documentation and JIRA cards for the Alt Lending Signal-Based Lead Generation System.

## Overview

An AI-powered platform that monitors public data sources for positive financial signals about delinquent student loan borrowers, enabling timely, context-aware outreach.

**Expected Impact:** 20-40% improvement in lead-to-recovery conversion rates
